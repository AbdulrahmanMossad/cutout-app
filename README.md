# CutOut AI Landing Page

A professional, SEO-optimized landing page for CutOut AI - a privacy-first offline background remover for Android.

## 🚀 Features

### Technical Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v3
- **Language**: TypeScript
- **UI Components**: Custom React components with smooth animations
- **Deployment**: Ready for Vercel

### SEO Optimization
- ✅ **Metadata**: Comprehensive meta tags and OG tags
- ✅ **Structured Data**: JSON-LD schema for SoftwareApplication
- ✅ **Sitemap**: Auto-generated XML sitemap
- ✅ **Robots**: SEO-friendly robots.txt
- ✅ **Manifest**: Progressive Web App manifest
- ✅ **Accessibility**: Semantic HTML, ARIA labels, alt text
- ✅ **Performance**: Optimized images, lazy loading, code splitting

### Design Features
- 🎨 **Modern Dark Theme**: Professional dark interface with blue/cyan accents
- 📱 **Fully Responsive**: Mobile-first design, works on all devices
- ✨ **Smooth Animations**: Fade-in, slide-in, and stagger animations
- 🎯 **Clear CTAs**: Google Play Store download buttons throughout
- 📊 **Feature Highlights**: 6 key features with icons
- 🔄 **How It Works**: Step-by-step process visualization
- 🛡️ **Trust Indicators**: Privacy badges and compliance markers

## 📁 Project Structure

```
.
├── app/
│   ├── layout.tsx           # Root layout with SEO metadata
│   ├── page.tsx             # Main landing page
│   ├── globals.css          # Global styles and animations
│   ├── sitemap.ts           # Dynamic sitemap generation
│   ├── robots.ts            # Robots.txt configuration
│   └── manifest.ts          # PWA manifest
├── components/
│   ├── header.tsx           # Navigation header with mobile menu
│   ├── hero.tsx             # Hero section with main CTA
│   ├── features.tsx         # 6-feature grid section
│   ├── how-it-works.tsx     # 4-step process visualization
│   ├── call-to-action.tsx   # Secondary CTA section
│   └── footer.tsx           # Footer with links and social
├── next.config.js           # Next.js configuration
├── tailwind.config.ts       # Tailwind CSS theme
├── tsconfig.json            # TypeScript configuration
└── package.json             # Dependencies
```
